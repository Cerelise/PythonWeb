## Flask+Echarts_Cov

### 基于Flask、Echarts的新冠病毒数据可视化项目

#### 使用方法

- 首先加载文件夹sql文件夹里mysql数据库文件，将数据导入数据库

-  加载完成之后，在终端中输入`python app.py`即可运行项目

- `spider.py`为疫情数据爬虫，运行爬虫则使用`python spider.py`语句

- `utils`为Flask连接数据库的文件，无需直接运行
